# Rust Tutorials

### [Install Rust][install-rust]
```shell
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

### Configure Shell
```shell
source "$HOME/.cargo/env"
```

### Build
```shell
cargo build
```

### Run
```shell
cargo run --bin basics *args
```

### Uninstall Rust
```shell
rustup self uninstall
```

[install-rust]: https://www.rust-lang.org/tools/install
